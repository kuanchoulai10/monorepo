    KServe Model Deployment (Hands-on)¶

    Prerequisites¶

    S3 Credential¶

    secret.yaml
    sa.yaml

    Install grpcurl¶

    brew install grpcurl

    Deploy Model¶

    inference-service-http.yaml
    

    kubectl apply -f inference-service-http.yaml
    inference-service-grpc.yaml
    

    kubectl apply -f inference-service-grpc.yaml

    Test Endpoints¶
