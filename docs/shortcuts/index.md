# Shortcuts

Keys and shortcuts for various tools. For a more comprehensive guide on how to use keyboard shortcuts extensions in Markdown, see:

[PyMdown Extensions](https://facelessuser.github.io/pymdown-extensions/extensions/keys/)

Here are some of the documented shortcuts that I use frequently:

- [`tmux`](./tmux.md)
- [`git`](./git.md)
- [`gh`]()

```bash
gh repo create <repo> \
  --source=. \
  --remote=origin \
  --push \
  --public

gh repo edit \
  --visibility public \
  --accept-visibility-change-consequences
```
