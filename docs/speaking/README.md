# 外語口說練習

在台灣當工程師當久了，即便都是在看英文的技術文件，還有演講影片，但實際上英文輸出的機會例如口說和寫作卻很少。為了提升自己的英文口說能力，我打算在2026年開始這項專案並記錄下來。

具體來說，我打算產出的是工程師在不同情境下的英文口說練習素材，包含以下幾種情境：

- 面試，包含行為面試和技術面試
- 技術簡報
- 日常工作會議
- 技術討論
- 閒聊

- Project Manager
- 2 Backend Engineers
- 2 Frontend Engineers
- 2 DevOps Engineers

- 1 Data Engineer
